* ‡πÄ‡∏ß‡πá‡∏ö‡πÑ‡∏ã‡∏ï‡πå **AQICN** ‡∏°‡∏µ API ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏î‡∏∂‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏Ñ‡∏∏‡∏ì‡∏†‡∏≤‡∏û‡∏≠‡∏≤‡∏Å‡∏≤‡∏®‡∏à‡∏≤‡∏Å‡∏™‡∏ñ‡∏≤‡∏ô‡∏µ‡∏ï‡πà‡∏≤‡∏á ‡πÜ ‡πÄ‡∏ä‡πà‡∏ô ‡∏™‡∏ñ‡∏≤‡∏ô‡∏µ‡∏Å‡∏£‡∏∏‡∏á‡πÄ‡∏ó‡∏û‡∏Ø (Bangkok) 
* ‡∏ï‡πâ‡∏≠‡∏á‡∏°‡∏µ ‚Äútoken‚Äù (API key) ‡∏ó‡∏µ‡πà‡∏•‡∏á‡∏ó‡∏∞‡πÄ‡∏ö‡∏µ‡∏¢‡∏ô‡∏Å‡∏±‡∏ö‡∏ó‡∏≤‡∏á Data Platform ‡∏Ç‡∏≠‡∏á WAQI ‡∏Å‡πà‡∏≠‡∏ô‡∏ñ‡∏∂‡∏á‡∏à‡∏∞‡πÄ‡∏£‡∏µ‡∏¢‡∏Å API ‡πÑ‡∏î‡πâ  

**‡πÇ‡∏Ñ‡πâ‡∏î‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡πÉ‡∏ô Python**

```python
import requests
import pandas as pd

def fetch_aqi_bangkok(station_id, token):
    # URL ‡∏Ç‡∏≠‡∏á API
    url = f"https://api.waqi.info/feed/@{station_id}/?token={token}"
    
    # ‡πÄ‡∏£‡∏µ‡∏¢‡∏Å API
    response = requests.get(url)
    if response.status_code != 200:
        raise Exception(f"Error fetching data: HTTP {response.status_code}")
    
    data = response.json()
    if data.get("status") != "ok":
        raise Exception(f"Error in API response: {data.get('data')}")
    
    # ‡∏î‡∏∂‡∏á‡∏Ñ‡πà‡∏≤‡∏ó‡∏µ‡πà‡∏™‡∏ô‡πÉ‡∏à
    aqi = data["data"].get("aqi")
    dominent_pollutant = data["data"].get("dominentpol")
    iaqi = data["data"].get("iaqi", {})
    pm25 = iaqi.get("pm25", {}).get("v")  # ‡∏ñ‡πâ‡∏≤‡∏°‡∏µ
    pm10 = iaqi.get("pm10", {}).get("v")
    time = data["data"].get("time", {}).get("s")
    
    # ‡∏™‡∏£‡πâ‡∏≤‡∏á DataFrame
    df = pd.DataFrame([{
        "station": "Bangkok",
        "time": time,
        "AQI": aqi,
        "Dominant Pollutant": dominent_pollutant,
        "PM2.5": pm25,
        "PM10": pm10
    }])
    
    return df

# Used to securely store your API key
from google.colab import userdata

# Replace 'WAQI_YOUR_TOKEN' with the name you used for the secret in Colab
WAQI_YOUR_TOKEN = userdata.get('WAQI_YOUR_TOKEN')

# Now you can use WAQI_YOUR_TOKEN in your code
print(WAQI_YOUR_TOKEN)

# WAQI_YOUR_TOKEN = "‡πÉ‡∏™‡πà‡πÇ‡∏ó‡πÄ‡∏Ñ‡πá‡∏ô‡∏Ç‡∏≠‡∏á‡∏Ñ‡∏∏‡∏ì‡∏ó‡∏µ‡πà‡∏ô‡∏µ‡πà"
station_id = "5773"  # Bangkok station (WAQI)
df_aqi = fetch_aqi_bangkok(station_id, WAQI_YOUR_TOKEN)
print(df_aqi)
```

```javascript
// ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡πÄ‡∏°‡∏∑‡πà‡∏≠‡∏°‡∏µ‡∏Å‡∏≤‡∏£‡πÄ‡∏£‡∏µ‡∏¢‡∏Å GET API
function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var rows = sheet.getDataRange().getValues();
  var headers = rows[0];
  var data = [];

  for (var i = 1; i < rows.length; i++) {
    var row = {};
    for (var j = 0; j < headers.length; j++) {
      row[headers[j]] = rows[i][j];
    }
    data.push(row);
  }

  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

// ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡πÄ‡∏°‡∏∑‡πà‡∏≠‡∏°‡∏µ‡∏Å‡∏≤‡∏£‡πÄ‡∏£‡∏µ‡∏¢‡∏Å POST API
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);

  // ‡πÄ‡∏û‡∏¥‡πà‡∏°‡πÅ‡∏ñ‡∏ß‡πÉ‡∏´‡∏°‡πà‡∏ï‡∏≤‡∏°‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏• JSON ‡∏ó‡∏µ‡πà‡∏™‡πà‡∏á‡∏°‡∏≤
  sheet.appendRow([data.name, data.age, data.city]);

  return ContentService
    .createTextOutput(JSON.stringify({status: "success", data: data}))
    .setMimeType(ContentService.MimeType.JSON);
}
```
