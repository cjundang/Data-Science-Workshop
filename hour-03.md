### 3. Storytelling with Data

Visualization ‡πÑ‡∏°‡πà‡πÉ‡∏ä‡πà‡πÅ‡∏Ñ‡πà‡∏Å‡∏≤‡∏£‡πÅ‡∏™‡∏î‡∏á‡∏Å‡∏£‡∏≤‡∏ü ‡πÅ‡∏ï‡πà‡∏Ñ‡∏∑‡∏≠‡∏Å‡∏≤‡∏£ **‡πÄ‡∏•‡πà‡∏≤‡πÄ‡∏£‡∏∑‡πà‡∏≠‡∏á‡∏à‡∏≤‡∏Å‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏• (Data Storytelling)** ‡∏ã‡∏∂‡πà‡∏á‡∏°‡∏µ‡∏´‡∏•‡∏±‡∏Å‡∏Å‡∏≤‡∏£‡∏Ñ‡∏∑‡∏≠:

1. **‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡∏õ‡∏£‡∏∞‡πÄ‡∏î‡πá‡∏ô‡∏´‡∏•‡∏±‡∏Å (Key Message):** ‡∏ï‡πâ‡∏≠‡∏á‡∏Å‡∏≤‡∏£‡πÉ‡∏´‡πâ‡∏ú‡∏π‡πâ‡∏ä‡∏°‡πÄ‡∏Ç‡πâ‡∏≤‡πÉ‡∏à‡∏≠‡∏∞‡πÑ‡∏£? ‡πÄ‡∏ä‡πà‡∏ô ‚Äú‡∏¢‡∏≠‡∏î‡∏Ç‡∏≤‡∏¢‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡πÉ‡∏ô Q3‚Äù
2. **‡πÄ‡∏•‡∏∑‡∏≠‡∏Å visualization ‡∏ó‡∏µ‡πà‡∏™‡∏ô‡∏±‡∏ö‡∏™‡∏ô‡∏∏‡∏ô‡∏Ç‡πâ‡∏≠‡∏Ñ‡∏ß‡∏≤‡∏°‡∏ô‡∏±‡πâ‡∏ô:** ‡πÄ‡∏ä‡πà‡∏ô line chart ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡πÄ‡∏•‡πà‡∏≤‡πÅ‡∏ô‡∏ß‡πÇ‡∏ô‡πâ‡∏°‡∏ï‡∏≤‡∏°‡πÄ‡∏ß‡∏•‡∏≤
3. **‡∏•‡∏î‡∏™‡∏¥‡πà‡∏á‡∏£‡∏ö‡∏Å‡∏ß‡∏ô (Decluttering):** ‡∏ï‡∏±‡∏î labels, gridlines ‡∏´‡∏£‡∏∑‡∏≠‡∏™‡∏µ‡∏ó‡∏µ‡πà‡πÑ‡∏°‡πà‡∏à‡∏≥‡πÄ‡∏õ‡πá‡∏ô‡∏≠‡∏≠‡∏Å
4. **‡πÄ‡∏ô‡πâ‡∏ô‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏™‡∏≥‡∏Ñ‡∏±‡∏ç (Highlighting):** ‡πÉ‡∏ä‡πâ‡∏™‡∏µ‡∏´‡∏£‡∏∑‡∏≠ annotation ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏î‡∏∂‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏ô‡πÉ‡∏à‡πÑ‡∏õ‡∏ó‡∏µ‡πà insight
5. **‡πÄ‡∏•‡πà‡∏≤‡πÄ‡∏õ‡πá‡∏ô‡πÄ‡∏£‡∏∑‡πà‡∏≠‡∏á‡∏£‡∏≤‡∏ß (Narrative):** ‡∏à‡∏≤‡∏Å introduction ‚Üí insight ‚Üí conclusion

 

## ‡∏ä‡∏∏‡∏î‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏™‡∏°‡∏°‡∏∏‡∏ï‡∏¥ (‡∏¢‡∏≠‡∏î‡∏Ç‡∏≤‡∏¢‡∏£‡∏≤‡∏¢‡∏™‡∏±‡∏õ‡∏î‡∏≤‡∏´‡πå‡∏Ç‡∏≠‡∏á‡∏õ‡∏µ‡∏´‡∏ô‡∏∂‡πà‡∏á ‡πÜ)

‡∏™‡∏°‡∏°‡∏∏‡∏ï‡∏¥‡∏ß‡πà‡∏≤‡πÄ‡∏£‡∏≤‡∏≠‡∏¢‡∏≤‡∏Å‡∏™‡∏∑‡πà‡∏≠‡∏™‡∏≤‡∏£‡∏õ‡∏£‡∏∞‡πÄ‡∏î‡πá‡∏ô‡∏´‡∏•‡∏±‡∏Å: **‚Äú‡∏¢‡∏≠‡∏î‡∏Ç‡∏≤‡∏¢‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡∏ä‡∏±‡∏î‡πÄ‡∏à‡∏ô‡πÉ‡∏ô‡πÑ‡∏ï‡∏£‡∏°‡∏≤‡∏™ 3 (Q3)‚Äù**

```python
import pandas as pd
import numpy as np

# ‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏£‡∏≤‡∏¢‡∏™‡∏±‡∏õ‡∏î‡∏≤‡∏´‡πå 52 ‡∏™‡∏±‡∏õ‡∏î‡∏≤‡∏´‡πå
rng = pd.date_range("2024-01-01", periods=52, freq="W")
np.random.seed(7)
base = np.linspace(100, 140, 52)                 # ‡πÅ‡∏ô‡∏ß‡πÇ‡∏ô‡πâ‡∏°‡πÇ‡∏î‡∏¢‡∏£‡∏ß‡∏°‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡πÄ‡∏•‡πá‡∏Å‡∏ô‡πâ‡∏≠‡∏¢
season = 10*np.sin(np.linspace(0, 6*np.pi, 52))  # ‡∏§‡∏î‡∏π‡∏Å‡∏≤‡∏•
noise = np.random.normal(0, 5, 52)
sales = base + season + noise

# ‡∏Å‡∏£‡∏∞‡∏ï‡∏∏‡πâ‡∏ô Q3 (‡∏™‡∏±‡∏õ‡∏î‡∏≤‡∏´‡πå ~27‚Äì39) ‡πÉ‡∏´‡πâ‡πÇ‡∏î‡∏î‡πÄ‡∏î‡πà‡∏ô
q3_idx = (rng.quarter == 3)
sales[q3_idx] += 15

df = pd.DataFrame({"week": rng, "sales": sales})
df["quarter"] = df["week"].dt.quarter
```


## 1) ‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡∏õ‡∏£‡∏∞‡πÄ‡∏î‡πá‡∏ô‡∏´‡∏•‡∏±‡∏Å (Key Message)

> ‚Äú‡∏¢‡∏≠‡∏î‡∏Ç‡∏≤‡∏¢‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡πÄ‡∏î‡πà‡∏ô‡∏ä‡∏±‡∏î‡πÉ‡∏ô Q3 ‡πÄ‡∏°‡∏∑‡πà‡∏≠‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏Å‡∏±‡∏ö Q1‚ÄìQ2 ‡πÅ‡∏•‡∏∞‡∏¢‡∏±‡∏á‡∏Ñ‡∏á‡∏ó‡∏£‡∏á‡∏ï‡∏±‡∏ß‡∏™‡∏π‡∏á‡πÉ‡∏ô Q4‚Äù

**‡∏ß‡∏¥‡∏ò‡∏µ‡∏Ñ‡∏¥‡∏î:** ‡∏õ‡∏£‡∏∞‡πÄ‡∏î‡πá‡∏ô‡∏ï‡πâ‡∏≠‡∏á‡∏™‡∏±‡πâ‡∏ô ‡∏ä‡∏±‡∏î‡πÄ‡∏à‡∏ô ‡∏ï‡∏≠‡∏ö‡∏Ñ‡∏≥‡∏ñ‡∏≤‡∏° ‚ÄúSo what?‚Äù‚Äî‡∏™‡∏¥‡πà‡∏á‡∏ó‡∏µ‡πà‡∏ú‡∏π‡πâ‡∏ä‡∏°‡∏Ñ‡∏ß‡∏£‡∏£‡∏±‡∏ö‡πÑ‡∏õ‡∏ï‡∏±‡∏î‡∏™‡∏¥‡∏ô‡πÉ‡∏à‡πÑ‡∏î‡πâ


 2) ‡πÄ‡∏•‡∏∑‡∏≠‡∏Å visualization ‡πÉ‡∏´‡πâ‡∏™‡∏≠‡∏î‡∏Ñ‡∏•‡πâ‡∏≠‡∏á‡∏Å‡∏±‡∏ö‡∏õ‡∏£‡∏∞‡πÄ‡∏î‡πá‡∏ô

‡πÄ‡∏û‡∏£‡∏≤‡∏∞‡πÄ‡∏õ‡πá‡∏ô **‡πÅ‡∏ô‡∏ß‡πÇ‡∏ô‡πâ‡∏°‡∏ï‡∏≤‡∏°‡πÄ‡∏ß‡∏•‡∏≤** ‚Üí ‡πÄ‡∏•‡∏∑‡∏≠‡∏Å **line chart** ‡πÄ‡∏õ‡πá‡∏ô‡∏Å‡∏£‡∏≤‡∏ü‡∏´‡∏•‡∏±‡∏Å ‡πÅ‡∏•‡∏∞‡πÄ‡∏™‡∏£‡∏¥‡∏°‡∏î‡πâ‡∏ß‡∏¢ **bar chart ‡πÅ‡∏ö‡∏ö‡∏™‡∏£‡∏∏‡∏õ‡πÄ‡∏â‡∏•‡∏µ‡πà‡∏¢‡∏£‡∏≤‡∏¢‡πÑ‡∏ï‡∏£‡∏°‡∏≤‡∏™** ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏¢‡∏∑‡∏ô‡∏¢‡∏±‡∏ô‡∏†‡∏≤‡∏û‡∏£‡∏ß‡∏°

**Line Chart: ‡πÅ‡∏ô‡∏ß‡πÇ‡∏ô‡πâ‡∏°‡∏£‡∏≤‡∏¢‡∏™‡∏±‡∏õ‡∏î‡∏≤‡∏´‡πå**

### 4. Visualization ‡πÄ‡∏õ‡πá‡∏ô‡πÄ‡∏Ñ‡∏£‡∏∑‡πà‡∏≠‡∏á‡∏°‡∏∑‡∏≠‡∏Ç‡∏≠‡∏á EDA

4.1 **‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö‡∏Ñ‡∏∏‡∏ì‡∏†‡∏≤‡∏û‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏• (Missing values, Outliers)**

**‡πÅ‡∏ô‡∏ß‡∏Ñ‡∏¥‡∏î**

* **Missing**: heatmap/‡∏ö‡∏≤‡∏£‡πå‡∏à‡∏≥‡∏ô‡∏ß‡∏ô NaN ‡∏ä‡πà‡∏ß‡∏¢‡∏£‡∏∞‡∏ö‡∏∏‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á/‡∏£‡∏π‡∏õ‡πÅ‡∏ö‡∏ö‡∏Ç‡∏≠‡∏á‡∏Å‡∏≤‡∏£‡∏´‡∏≤‡∏¢
* **Outliers**: boxplot/violin plot ‡∏ä‡πà‡∏ß‡∏¢‡πÄ‡∏´‡πá‡∏ô‡∏Ñ‡πà‡∏≤‡∏ú‡∏¥‡∏î‡∏õ‡∏Å‡∏ï‡∏¥‡πÅ‡∏•‡∏∞‡∏Å‡∏≤‡∏£‡∏Å‡∏£‡∏∞‡∏à‡∏≤‡∏¢

**‡∏™‡∏≤‡∏ò‡∏¥‡∏ï**

```python
import pandas as pd, numpy as np, seaborn as sns, matplotlib.pyplot as plt
np.random.seed(0)

# ‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏™‡∏°‡∏°‡∏∏‡∏ï‡∏¥
n = 200
df = pd.DataFrame({
    "sales": np.random.normal(100, 15, n),

})
```

**‡πÅ‡∏õ‡∏•‡∏ú‡∏•**: ‡∏à‡∏∏‡∏î‡∏™‡∏ß‡πà‡∏≤‡∏á‡πÉ‡∏ô heatmap ‡∏ö‡∏≠‡∏Å‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå/‡πÅ‡∏ñ‡∏ß‡∏ó‡∏µ‡πà‡∏´‡∏≤‡∏¢‡∏ö‡πà‡∏≠‡∏¢; ‡∏´‡∏≤‡∏á‡∏¢‡∏≤‡∏ß/‡∏à‡∏∏‡∏î‡∏ô‡∏≠‡∏Å whisker ‡πÉ‡∏ô boxplot ‡∏Ñ‡∏∑‡∏≠ outliers ‡∏ó‡∏µ‡πà‡∏ï‡πâ‡∏≠‡∏á‡∏û‡∏¥‡∏à‡∏≤‡∏£‡∏ì‡∏≤ (‡∏¢‡∏∑‡∏ô‡∏¢‡∏±‡∏ô‡∏î‡πâ‡∏ß‡∏¢‡∏™‡∏ñ‡∏¥‡∏ï‡∏¥/‡∏ö‡∏£‡∏¥‡∏ö‡∏ó‡∏ò‡∏∏‡∏£‡∏Å‡∏¥‡∏à‡∏Å‡πà‡∏≠‡∏ô‡∏ï‡∏±‡∏î‡∏™‡∏¥‡∏ô‡πÉ‡∏à‡∏•‡∏ö‡∏´‡∏£‡∏∑‡∏≠‡∏ï‡∏£‡∏∂‡∏á‡∏Ñ‡πà‡∏≤)

**‡πÅ‡∏ô‡∏ß‡∏Ñ‡∏¥‡∏î**

* **Correlation**: heatmap ‡∏Ç‡∏≠‡∏á‡∏Ñ‡πà‡∏≤‡∏™‡∏´‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå; pairplot ‡∏î‡∏π‡∏£‡∏π‡∏õ‡∏ó‡∏£‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå

**‡∏™‡∏≤‡∏ò‡∏¥‡∏ï (‡πÄ‡∏ß‡∏•‡∏≤ & ‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå)**

```python
# ‡∏™‡∏£‡πâ‡∏≤‡∏á time series ‡∏°‡∏µ‡πÅ‡∏ô‡∏ß‡πÇ‡∏ô‡πâ‡∏°+‡∏§‡∏î‡∏π‡∏Å‡∏≤‡∏•
rng = pd.date_range("2024-01-01", periods=180, freq="D")

noise = np.random.normal(0, 4, len(rng))

# Line + rolling mean
tsa["roll7"] = tsa["y"].rolling(7, center=True).mean()
sns.lineplot(data=tsa, x="date", y="y", label="daily")
sns.lineplot(data=tsa, x="date", y="roll7", label="7-day mean")

# ‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå‡∏Å‡∏±‡∏ô (sales-visits) ‡∏î‡πâ‡∏ß‡∏¢ heatmap

# ‡∏î‡∏π‡∏´‡∏•‡∏≤‡∏¢‡∏°‡∏¥‡∏ï‡∏¥‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏£‡∏ß‡∏î‡πÄ‡∏£‡πá‡∏ß
```

**‡πÅ‡∏õ‡∏•‡∏ú‡∏•**: ‡πÄ‡∏™‡πâ‡∏ô rolling ‡πÄ‡∏ú‡∏¢‡πÅ‡∏ô‡∏ß‡πÇ‡∏ô‡πâ‡∏°‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡πÅ‡∏•‡∏∞‡∏£‡∏π‡∏õ‡πÅ‡∏ö‡∏ö‡∏Ç‡∏∂‡πâ‡∏ô‡∏•‡∏á‡∏õ‡∏£‡∏∞‡∏à‡∏≥‡∏™‡∏±‡∏õ‡∏î‡∏≤‡∏´‡πå; heatmap ‡πÉ‡∏´‡πâ‡∏Ñ‡πà‡∏≤‡∏™‡∏±‡∏°‡∏õ‡∏£‡∏∞‡∏™‡∏¥‡∏ó‡∏ò‡∏¥‡πå‡∏Ñ‡∏ß‡∏≤‡∏°‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ô‡∏ò‡πå‡∏ß‡πà‡∏≤‡∏ö‡∏ß‡∏Å/‡∏•‡∏ö/‡∏≠‡πà‡∏≠‡∏ô‡πÅ‡∏£‡∏á‡πÄ‡∏û‡∏µ‡∏¢‡∏á‡πÉ‡∏î; pairplot ‡∏ä‡πà‡∏ß‡∏¢‡πÄ‡∏´‡πá‡∏ô nonlinearity ‡∏´‡∏£‡∏∑‡∏≠‡∏Å‡∏•‡∏∏‡πà‡∏°‡∏¢‡πà‡∏≠‡∏¢

 
4.3 **‡∏™‡∏∑‡πà‡∏≠‡∏™‡∏≤‡∏£‡∏Å‡∏±‡∏ö‡∏ú‡∏π‡πâ‡∏≠‡∏∑‡πà‡∏ô‡πÑ‡∏î‡πâ‡∏á‡πà‡∏≤‡∏¢ (Communication)**

**‡πÅ‡∏ô‡∏ß‡∏Ñ‡∏¥‡∏î**
