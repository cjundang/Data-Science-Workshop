### 3. Storytelling with Data

Visualization р╣Др╕бр╣Ир╣Гр╕Кр╣Ир╣Бр╕Др╣Ир╕Бр╕▓р╕гр╣Бр╕кр╕Фр╕Зр╕Бр╕гр╕▓р╕Я р╣Бр╕Хр╣Ир╕Др╕╖р╕нр╕Бр╕▓р╕г **р╣Ар╕ер╣Ир╕▓р╣Ар╕гр╕╖р╣Ир╕нр╕Зр╕Ир╕▓р╕Бр╕Вр╣Йр╕нр╕бр╕╣р╕е (Data Storytelling)** р╕Лр╕╢р╣Ир╕Зр╕бр╕╡р╕лр╕ер╕▒р╕Бр╕Бр╕▓р╕гр╕Др╕╖р╕н:

1. **р╕Бр╕│р╕лр╕Щр╕Фр╕Ыр╕гр╕░р╣Ар╕Фр╣Зр╕Щр╕лр╕ер╕▒р╕Б (Key Message):** р╕Хр╣Йр╕нр╕Зр╕Бр╕▓р╕гр╣Гр╕лр╣Йр╕Ьр╕╣р╣Йр╕Кр╕бр╣Ар╕Вр╣Йр╕▓р╣Гр╕Ир╕нр╕░р╣Др╕г? р╣Ар╕Кр╣Ир╕Щ тАЬр╕вр╕нр╕Фр╕Вр╕▓р╕вр╣Ар╕Юр╕┤р╣Ир╕бр╕Вр╕╢р╣Йр╕Щр╣Гр╕Щ Q3тАЭ
2. **р╣Ар╕ер╕╖р╕нр╕Б visualization р╕Чр╕╡р╣Ир╕кр╕Щр╕▒р╕Ър╕кр╕Щр╕╕р╕Щр╕Вр╣Йр╕нр╕Др╕зр╕▓р╕бр╕Щр╕▒р╣Йр╕Щ:** р╣Ар╕Кр╣Ир╕Щ line chart р╣Ар╕Юр╕╖р╣Ир╕нр╣Ар╕ер╣Ир╕▓р╣Бр╕Щр╕зр╣Вр╕Щр╣Йр╕бр╕Хр╕▓р╕бр╣Ар╕зр╕ер╕▓
3. **р╕ер╕Фр╕кр╕┤р╣Ир╕Зр╕гр╕Ър╕Бр╕зр╕Щ (Decluttering):** р╕Хр╕▒р╕Ф labels, gridlines р╕лр╕гр╕╖р╕нр╕кр╕╡р╕Чр╕╡р╣Ир╣Др╕бр╣Ир╕Ир╕│р╣Ар╕Ыр╣Зр╕Щр╕нр╕нр╕Б
4. **р╣Ар╕Щр╣Йр╕Щр╕Вр╣Йр╕нр╕бр╕╣р╕ер╕кр╕│р╕Др╕▒р╕Н (Highlighting):** р╣Гр╕Кр╣Йр╕кр╕╡р╕лр╕гр╕╖р╕н annotation р╣Ар╕Юр╕╖р╣Ир╕нр╕Фр╕╢р╕Зр╕Др╕зр╕▓р╕бр╕кр╕Щр╣Гр╕Ир╣Др╕Ыр╕Чр╕╡р╣И insight
5. **р╣Ар╕ер╣Ир╕▓р╣Ар╕Ыр╣Зр╕Щр╣Ар╕гр╕╖р╣Ир╕нр╕Зр╕гр╕▓р╕з (Narrative):** р╕Ир╕▓р╕Б introduction тЖТ insight тЖТ conclusion

 

## р╕Кр╕╕р╕Фр╕Вр╣Йр╕нр╕бр╕╣р╕ер╕кр╕бр╕бр╕╕р╕Хр╕┤ (р╕вр╕нр╕Фр╕Вр╕▓р╕вр╕гр╕▓р╕вр╕кр╕▒р╕Ыр╕Фр╕▓р╕лр╣Мр╕Вр╕нр╕Зр╕Ыр╕╡р╕лр╕Щр╕╢р╣Ир╕З р╣Ж)

р╕кр╕бр╕бр╕╕р╕Хр╕┤р╕зр╣Ир╕▓р╣Ар╕гр╕▓р╕нр╕вр╕▓р╕Бр╕кр╕╖р╣Ир╕нр╕кр╕▓р╕гр╕Ыр╕гр╕░р╣Ар╕Фр╣Зр╕Щр╕лр╕ер╕▒р╕Б: **тАЬр╕вр╕нр╕Фр╕Вр╕▓р╕вр╣Ар╕Юр╕┤р╣Ир╕бр╕Вр╕╢р╣Йр╕Щр╕Кр╕▒р╕Фр╣Ар╕Ир╕Щр╣Гр╕Щр╣Др╕Хр╕гр╕бр╕▓р╕к 3 (Q3)тАЭ**

```python
import pandas as pd
import numpy as np

# р╕кр╕гр╣Йр╕▓р╕Зр╕Вр╣Йр╕нр╕бр╕╣р╕ер╕гр╕▓р╕вр╕кр╕▒р╕Ыр╕Фр╕▓р╕лр╣М 52 р╕кр╕▒р╕Ыр╕Фр╕▓р╕лр╣М
rng = pd.date_range("2024-01-01", periods=52, freq="W")
np.random.seed(7)
base = np.linspace(100, 140, 52)                 # р╣Бр╕Щр╕зр╣Вр╕Щр╣Йр╕бр╣Вр╕Фр╕вр╕гр╕зр╕бр╣Ар╕Юр╕┤р╣Ир╕бр╕Вр╕╢р╣Йр╕Щр╣Ар╕ер╣Зр╕Бр╕Щр╣Йр╕нр╕в
season = 10*np.sin(np.linspace(0, 6*np.pi, 52))  # р╕др╕Фр╕╣р╕Бр╕▓р╕е
noise = np.random.normal(0, 5, 52)
sales = base + season + noise

# р╕Бр╕гр╕░р╕Хр╕╕р╣Йр╕Щ Q3 (р╕кр╕▒р╕Ыр╕Фр╕▓р╕лр╣М ~27тАУ39) р╣Гр╕лр╣Йр╣Вр╕Фр╕Фр╣Ар╕Фр╣Ир╕Щ
q3_idx = (rng.quarter == 3)
sales[q3_idx] += 15

df = pd.DataFrame({"week": rng, "sales": sales})
df["quarter"] = df["week"].dt.quarter
```
